Goals

We set out to program the Anki Cozmo robot to be able to localize itself rotationally using Monte Carlo localization. As such, we should be able to do the following: have the robot familiarize itself with its environment, including noting a specified “home location,” rotate it arbitrarily such that it is no longer facing the home position, and the robot should be able to rotate itself back to the home position after doing some sort of process to figure out where it is.

Set up

Materials Needed

  • 1 Anki Cozmo robot
  • 1 Iphone
  • At least one computer with the below software installed on it with a way of connecting to the Iphone
  • A cable to connect the phone to the computer, probably micro USB or USB-C

Software Prerequisites

  • The Cozmo app
  • Python 3.7
  • Visual Studio Code
    • Jupyter Notebooks extension
  • Python libraries to import:
    • Numpy
    • openCV
    • Math
    • random
    • Cozmo
    • MatPlotLib.pyplot
    • Os
    • sys

Setup Instructions

  • For each computer you would like to be able to actually run programs on the robot from, you must install the android debug bridge and the cozmo SDK
  • If Python is not installed already, go to https://www.python.org/downloads/ and download the latest version available, or at least 3.10.
  • Download and install VSCode from the link https://code.visualstudio.com/download. We preferred to use VSCode for our project since it has particularly good Jupyter Notebook integrations.
